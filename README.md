# DAO-Proposal-Analyzer-Debate-Agent
LangGraph multi-agent system that automates DAO governance research. Feed it a Snapshot or Tally proposal and three agents go to work an Advocate argues for it, a Critic argues against it, and a Judge synthesizes the debate into a clear vote recommendation.
